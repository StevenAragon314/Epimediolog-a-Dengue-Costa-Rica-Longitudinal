# Epimediolog-a-Dengue-Costa-Rica-Longitudinal
Desarrollo de indicadores para la incidencia del Dengue (2020-2026) en Costa Rica. Integra datos longitudinales de casos regionales con variables climáticas, demográficas, poblacionales y campañas para optimizar la vigilancia epidemiológica y la toma de decisiones en salud pública.
